# Mission Control

This repo is the **intake + operating system** for agent-driven execution across multiple repositories.

## The Board
- Project: https://github.com/users/basementbrain10-bot/projects/1
- Status: Inbox → Next → In Progress → Blocked → Done

## How to use
1. Create an issue (Task/Bug/Research)
2. Add it to the Mission Control Project
3. Agents execute work in the correct target repo, then link PRs/issues back here.

## Conventions
- Prefix issue titles with: `[TASK]`, `[BUG]`, `[RESEARCH]`
- Keep acceptance criteria explicit.
